       - `execute_sell`: Queries PortfolioPosition for open position matching rec.ticker, if not found returns failure. Otherwise fetches current price, calculates PnL, updates PortfolioState.cash += proceeds, closes position (status="closed", exit_price, realized_pnl), creates TradeRecord(action="SELL", reasoning="[PAPER] " + rec.reasoning), commits, returns ExecutionResult(success=True, ...).
       - For options sell, use the same options pricing fallback pattern from portfolio_manager.py (entry_price if chain unavailable).

    Follow project conventions:

    - Single-line docstrings
    - Relative imports within quant_team package
    - `logger = logging.getLogger("quant_team")`
    - Log each paper trade: `logger.info(f"[PAPER] {action} {quantity} {ticker} @ ${price:.2f}")`
